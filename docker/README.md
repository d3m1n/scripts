## HOW TO USE
```bash
curl -s "https://raw.githubusercontent.com/d3m1n/scripts/main/docker/install-docker.sh" | bash -s
```

If you need to remove conflicting packages before install:
```bash
curl -s "https://raw.githubusercontent.com/d3m1n/scripts/main/docker/install-docker.sh" | bash -s -- --remove-conflicts
```
